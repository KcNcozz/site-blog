## 开发环境配置
1. VSCode 下载地址：https://code.visualstudio.com/ 插件用微软账户同步
2. Node.js 安装教程：https://lino-ai.blog.csdn.net/article/details/155107677
3. Git 下载地址：https://git-scm.com/install/windows
    - 可设置为中文界面
4. Docker (使用Windows自带的WSL2) 
    - 需要开启Windows的Linux子系统
    - 下载Docker Desktop：https://www.docker.com/products/docker-desktop
    - 安装Docker Desktop 一路下一步安装即可
    - 在docker中安装具体镜像详见**语雀文档**
5. Java环境配置
6. JetBrains 直接安装JetBrains Toolbox 需要调整软件的默认安装位置
7. 影视相关软件：达芬奇、PR、AE、剪映 （使用准备好的安装包）
