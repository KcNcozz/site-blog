Typescript ​

1. 数据类型 ​

基础类型 ​

同javascript一致

任意类型 ​

  • any 任意类型
  • unknown 未知类型

数据类型顺序：

  1. any unknown
  2. Object
  3. Number String Boolean
  4. number string boolean
  5. '123' 123
  6. never

unknown只能赋值给自身或者any，unknown类型无法去读属性，方法也不可以调用，因此unknown比any更安全。

2. Object、object、{} ​

Object：所有的原始类型和对象类型都指向Object。

object：object类型只能是引用类型，不能是原始类型。 例如：
123 '123' true ❌️
{} [] () => 123 ✅️

{}：同Object，可以理解为new Object()

    } else {
