安装 ​

  1. 安装命令: npm install -g @nestjs/cli
  2. 创建项目: nest new project-name可以简写为nest n project-name
    • 跳过安装git: nest new project-name -g
    • 跳过安装依赖: nest new project-name -s
  3. 启动项目: nest start

IoC和DI ​

  • IoC (Inversion of Control，控制反转)：对象控制权交给框架, 即对象的创建、依赖管理、生命周期控制，不再由业务代码自己负责，而是交给框架或容器负责。
  • DI (Dependency Injection，依赖注入)：框架把依赖自动注入给你
  • AOP：把日志、权限、异常等通用逻辑从业务代码中抽离出来

在 NestJS 中，@Injectable() + constructor 注入 就是 DI，Guard / Interceptor / Pipe / Filter 就是典型 AOP。

When it returns a JavaScript primitive type (e.g., string, number, boolean), however, Nest will send just the value without attempting to serialize it. ts

  const app = await NestFactory.create(AppModule);
  app.useGlobalPipes(
    new ValidationPipe({
      transform: true, // 添加转换
      whitelist: true, // 未经验证的字段会被过滤掉
      forbidNonWhitelisted: true, // 未经验证字段会报错
    }),
  );
  await app.listen(process.env.PORT ?? 3000);
}
